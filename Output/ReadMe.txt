Before first run you must open the setup application and configure your 
graphics and sound preferences. Then configure keyboard control.

All configurations and high scores are saved in the User's profile folder:

c:\Users\$USER\AppData\Roaming\GadzPl\Retrofire\

You can freely delete it to reset the configuration or clear the saved game 
states.
